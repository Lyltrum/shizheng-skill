# 时政简报生成规则

把多源内容浓缩成一份便于快速阅读的**当日时政简报**。目标：用户花 1～2 分钟就能掌握当天时政要点。

## 取材

- **主干 = 新闻联播**（`xinwenlianbo` 源的 `items`）：时政密度最高、最权威，作为"要闻"骨架。
- **补充 = 央视国内要闻**（`cctv-china` 源的 `items`）：补充联播之外、全天滚动的时政/政务动态。
- 两源若有同一事件，合并成一条，不重复罗列。
- 央视要闻里明显非时政的内容（纯民生趣闻、文体、天气等）可略去或仅一句带过——本简报聚焦**时政**。

## 分组

按主题归类，只列当天实际出现的类别：

- **时政要闻** — 领导人活动、重要会议、重大部署、人事
- **政策政务** — 政策文件发布、部委举措、督查巡查、政务改革
- **外交** — 元首外交、双边/多边、外交部表态、国际表态
- **其它时政相关** — 不便归入上面的时政内容

## 每条要点

- 一条压成 1～2 句话，保留：谁、做了什么、关键结论/数字。
- 去掉套话和过渡句，只留信息密度高的内容。
- **标注来源与出处**：
  - 来自联播：句末标 `[联播N]`（N=该条 index），便于用户追问全文。
  - 来自央视要闻：句末附该条 `url`。

## 格式示例

```
时政简报 · 2026年6月10日（基于 6月9日新闻联播 + 央视实时要闻）

【时政要闻】
- 习近平结束对朝鲜国事访问回到北京，访问期间与金正恩达成多项共识…… [联播7]
- 中央安全生产考核巡查受理问题隐患线索工作启动 https://news.cctv.com/...

【政策政务】
- 五部门出台多项改革举措，便利企业办理不动产登记…… https://news.cctv.com/...

【外交】
- 外交部：绝不允许日菲侵害中方海洋权益 https://news.cctv.com/...

来源：新闻联播（<联播url>）+ 央视国内要闻
```

## 注意

- 不臆测、不补充源数据之外的信息；数字、人名、地名以原文为准。
- 客观中立转述，不加政治评论或立场延伸。
